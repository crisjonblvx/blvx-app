- Skeleton loading states
- Axios API calls with credentials

Follow existing BLVX patterns:

```jsx
// Theme detection
const isDark = document.documentElement.getAttribute('data-theme') === 'dark';

// Status colors
const statusColors = {
  online: 'bg-green-500',

const textClass = isDark ? 'text-white' : 'text-gray-900';
```

## API Reference

| Endpoint | Returns | Notes |
|----------|---------|-------|
| `GET /users/online?limit=50` | `{ total_online, users[] }` | Active in last 30 min |
| `GET /users/discover?limit=20` | `user[]` with `suggestion_reason` | Smart suggestions |
| `GET /users/trending?limit=20` | `user[]` with `trending_score` | This week's risers |
| `GET /users/{id}/connections` | Mutual info, voucher, status | Profile context |
| `GET /users/activity/friends?limit=30` | `activity[]` | Friend actions |
